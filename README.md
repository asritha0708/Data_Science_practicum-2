## Dataset

This project uses the **DEAP Dataset** - A Database for Emotion Analysis using Physiological Signals.

- 32 participants (s01.dat >  s32.dat)
- 32 EEG channels at 128Hz sampling rate
- 40 trials per subject
- Labels: Valence, Arousal, Dominance, Liking (scores 1–9)

> The DEAP dataset requires registration to download from the official source. The preprocessed data used in this project is available via the link below.

 **Download Dataset (Google Drive):**
[Click Here to Download](https://drive.google.com/drive/folders/1ytYDJbl_kgGe-mtcD2aygFeO3Qphx9Yr?usp=drive_link)

Once downloaded, place all `.dat` files inside a folder named `data_preprocessed_python/` in the root directory of this project.
